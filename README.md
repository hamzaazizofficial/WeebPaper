# WeebPaper
Android App based on MVVM Architecture which shows random anime images from some famous subreddits like **r/animeArt**, **r/imaginarySliceOfLife**, **r/animePhoneWallpapers** and **animeWallpapersSfw**.

## Features
- Save Wallpaper
- Set Wallaper
- Share Wallpaper
- Select Subreddit 
- Zoom Image
